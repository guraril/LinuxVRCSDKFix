# Linux VRCSDK Fix

## 機能

### VRChat Avatar

- Build & Testを実行できるように

### VRChat World

- ワールドをアップロードできない問題の修正
- Build & Testを実行できるように

## 使い方

### Build & Test

1. Unity上部のメニューからTools/Fix VRCWorld Issues/Open Control Panelをクリック
2. 出現したウィンドウの下部にあるBuild & Testボタンをクリック

これだけで簡単にVRChatのローカルテストができます！
